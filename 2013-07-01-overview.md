# Proposed overview of a holding ontology

## Relation between Item and Item
``` {.ditaa}
                           +-----------+
                   +-------+           +-------+
narrowerExemplar / |       |   Item    |       | broaderExemplar /
narrowerExemplarOf |       |           |       | broaderExemplarOf
                   +------>|           |<------+
                           +-----------+

```

## Relation between Item and Documents
``` {.ditaa}
+---------------------+    exemplarOf   +-------+    foaf:page    +-----------+
|      Document       |<----------------+       +---------------->| Document  |
| (Title Description) |                 | Item  |                 | (Website) |
|                     |---------------->|       |<----------------+           |
+---------------------+     exemplar    +-------+    foaf:topic   +-----------+
                                            ^
                                            |
                                    foaf:primaryTopic /
                                    foaf:primaryTopicOf
                                            |
                                            v
                                 +-----------------------+
                                 |      Document         |
                                 | (Holding Description) |
                                 +-----------------------+
```

+---------+     heldBy   +--------+   daia:availableFor /  +---------+
|         |<-------------+        |   daia:unavailableFor  |         |
|  Agent  +------------->|  Item  +----------------------->| Service |
|         |     holds    |        |<-----------------------+         |
+----+----+              +--------+   daia:availableOf /   +---------+
     |                                daia:unavailableOf        ^
     |                                                          |
     |                       ssso:provides                      |
